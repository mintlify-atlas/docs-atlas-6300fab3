<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-semibold dark:text-white text-gray-900 mb-4">Quick Start</h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">Get up and running with the Service Orders Management System in just a few steps.</p>

  <Steps>
    <Step title="Install Dependencies">
      Clone the repository and install the required dependencies using your preferred package manager.

      <CodeGroup>
      ```bash npm
      npm install
      ```
      ```bash yarn
      yarn install
      ```
      ```bash pnpm
      pnpm install
      ```
      </CodeGroup>
    </Step>

    <Step title="Configure Environment">
      Set up your API base URL in the Nuxt configuration. The system connects to a backend API for authentication and data management.

      ```typescript nuxt.config.ts
      export default defineNuxtConfig({
        runtimeConfig: {
          public: {
            apiBaseUrl: 'http://localhost:8000/api/'
          }
        }
      })
      ```

      <Note>
        Update the `apiBaseUrl` to point to your backend API endpoint.
      </Note>
    </Step>

    <Step title="Start Development Server">
      Launch the development server to start building and testing your application.

      ```bash
      npm run dev
      ```

      The application will be available at `http://localhost:3000`.
    </Step>

    <Step title="Authenticate">
      Access the login page and authenticate with your credentials to start managing service orders.

      <Tip>
        The authentication system uses JWT tokens stored in localStorage for session management.
      </Tip>
    </Step>
  </Steps>
</div>
